# 标签体系（Labels）

建议在仓库 Settings → Labels 创建以下标签（用于 Quest 元数据与计分）：

## 分值（Points）

- `Points: 10`
- `Points: 50`
- `Points: 100`

> 计分时会从 Issue Labels 里读取 `Points: XX` 作为本次 `/award` 的加分值。

## 类型（Quest Type）

- `Quest: Learning`（学习）
- `Quest: Coding`（开发）
- `Quest: Promotion`（推广）

## 状态（Status）

- `Status: Open`（进行中）
- `Status: Completed`（已完成）
