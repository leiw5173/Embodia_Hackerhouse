## 参与方式

1. **接任务**：在 Issues 里找到感兴趣的 Quest
2. **做任务**：学习 / 开发 / 推广
3. **交任务**：
   - **代码类**：提交 PR，并在描述里关联 Issue（例如 `Fixes #12`）
   - **非代码类**：在对应 Issue 下评论截图/链接（文章链接、视频链接等）
4. **计分**：由维护者在 Issue 下回复 **`/award @你的用户名`** 发放积分（分值从该 Issue 的 `Points: XX` 标签读取）
