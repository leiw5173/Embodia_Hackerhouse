# Embodia Hackerhouse（魔珐星云黑客营）

本仓库是一个 **IssueOps/ChatOps** 形式的“黑客营任务板 + 积分榜”。

更多说明文档：

- `docs/participation.md`（参与方式）
- `docs/chatops.md`（/award 指令与计分规则）
- `docs/labels.md`（标签体系）
- `docs/quests.md`（Quest 编写规范）
- `docs/maintainers.md`（维护者核销与运营流程）

---

## 参与方式

1. **接任务**：在 Issues 里找到感兴趣的 Quest
2. **做任务**：学习 / 开发 / 推广
3. **交任务**：
   - **代码类**：提交 PR，并在描述里关联 Issue（例如 `Fixes #12`）
   - **非代码类**：在对应 Issue 下评论截图/链接（文章链接、视频链接等）
4. **计分**：由维护者在 Issue 下回复 **`/award @你的用户名`** 发放积分（分值从该 Issue 的 `Points: XX` 标签读取）

---

## 🏆 开发者荣誉榜（自动更新）

> 排行榜会在 **Issue 被关闭** 或 **PR 被合并/关闭** 时由 GitHub Actions 自动刷新。
> 计分规则：只统计 **已关闭（closed）且带 `Points: XX` 标签** 的 Issue；优先按 **Assignees** 记分（多人会均分）。
